### Docker 환경에서 maria DB 사용

*docker image 내의 mysql client가 설치되지 않아서 발생하는 이슈*
- 해결 1) mariadb 명령어로 접속한다.
```
mariadb -uroot -p
```

- 해결 2) docker 이미지내에 mysql client를 설치한다.
```
apt-get update
apt-get install mysql-client
```
